7 Reverse Integer

每次将最后一位*10+之前的一位。

165 Compare Version Numbers

首先把.去掉，取两个字符串中最大的长度，然后逐位比较。

66 Plus One

如果该位小于9就+1返回，否则=0.这样可以完成进位。

8 String to Integer (atoi)

首先，handle空字串，去掉空格后再handle，否则会越界。后面当ans越界时返回正无穷或负无穷。ans每次*10再加上tmp。

258 Add Digits

返回9或对9取余的结果即可。
